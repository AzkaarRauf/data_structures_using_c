Sorting

16th-May guidance: "Go with the quick, merge, or insertion sort only — these are the important ones." Case studies come "mostly from searching and sorting." Learn insertion sort cold; know the idea of quick & merge.

Sorting = arranging elements in order (ascending or descending). We'll use the list 23 78 45 8 32 56 throughout (the lecturer's example).

1. Selection Sort (the easy one)

Repeatedly find the smallest remaining element and swap it to the front.

2. Insertion Sort (learn this cold)

Like sorting playing cards in your hand: take the next card and slide it back into its correct place among the cards already sorted.

Grow a sorted block on the left. Take each next element, shift bigger ones right, drop it into place.

StepSorted block | rest
start23 | 78 45 8 32 56
insert 7823 78 | 45 8 32 56
insert 4523 45 78 | 8 32 56
insert 88 23 45 78 | 32 56
insert 328 23 32 45 78 | 56
insert 568 23 32 45 56 78 |
void insertionSort(int a[], int n){
    int i, j, temp;
    for(i = 1; i < n; i++){
        temp = a[i];                    // element to place
        j = i - 1;
        while(j >= 0 && a[j] > temp){    // shift bigger ones right
            a[j+1] = a[j];
            j--;
        }
        a[j+1] = temp;                  // drop into the gap
    }
}

3. Quick Sort (know the idea)

  • Pick a pivot. Partition: smaller elements to its left, bigger to its right.
  • The pivot is now in its final place. Recurse on the left and right parts.
  • Also called partition-exchange sort. Average complexity O(n log n).

4. Merge Sort (know the idea)

  • Divide the array into two halves, sort each half (recursively).
  • Merge the two sorted halves into one sorted array.
  • Complexity O(n log n) in all cases.

5. Scoring in the exam

If asked to "write the selection/insertion sort code," you only need the sort function (the few lines above) + a short explanation of how it works — not a whole program with input scaffolding. Practice the trace tables; the examiner may give a list and ask for the array after pass 2.
