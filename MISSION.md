# Mission: Data Structures Using C

## Why
Pass BCA semester exams on Data Structures Using C. Need to understand concepts well enough to write correct theory answers and implement DS programs in C under exam conditions.

## Success looks like
- Explain any standard DS (array, linked list, stack, queue, tree, graph) clearly in theory
- Write C programs for DS operations (insert, delete, traverse, search) from memory
- Solve standard exam questions: trace algorithms, find time complexity, draw diagrams
- Implement sorting and searching algorithms in C

## Constraints
- Knows C basics: loops, functions, pointers, structs — no need to teach C syntax
- Focus on exam-relevant depth, not production-level engineering
- Primary resource: provided PDF textbook

## Out of scope
- Advanced algorithms beyond BCA syllabus (dynamic programming, segment trees, etc.)
- C++ STL, templates, or object-oriented approaches
- Competitive programming optimizations
