# C Basics Established as Prior Knowledge

User knows C fundamentals: loops, functions, pointers, structs. DS is entirely new. This means lessons can skip C syntax explanations and go straight to using these constructs to implement data structures. Start with the DS concept, then show the C code — no need to explain `int`, `for`, `malloc` from scratch.

**Implications:** First lesson can immediately use `struct` for DS definitions and pointer manipulation for dynamic structures. Zone of proximal development starts at DS concepts + C implementations, not C language itself.
