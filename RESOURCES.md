# Data Structures Using C — Resources

## Knowledge

- **Primary Textbook (SOURCE OF TRUTH)** — `Data Structure Using W FIN (1).pdf` (Amity SLM, 90 pages)
  Core source. All lessons align to it. Extracted to `pdf_extracted.txt` (via `pdftotext -layout`) — grep that to verify facts. Modules: I Intro+Arrays, II Stacks+Queues, III Linked Lists, IV Trees+Graphs, V Searching+Sorting. Has end-of-module MCQs for practice.

- **Lecture Transcripts** — `lectures/transcripts/*.txt` (4 classes: 11 Apr, 12 Apr, 19 Apr, 16 May)
  Signal for WHAT is exam-important (not truth). Distilled into `reference/exam-blueprint.html`.

- [Data Structures — GeeksForGeeks](https://www.geeksforgeeks.org/data-structures/)
  Excellent reference for C implementations with diagrams. Use for: code examples, algorithm traces, complexity tables.

- [CS50 — Harvard (YouTube)](https://www.youtube.com/c/cs50)
  Highly visual explanations. Use for: understanding memory, pointers, linked lists with animations.

- [NPTEL — Data Structures and Algorithms (IIT)](https://nptel.ac.in/courses/106/102/106102064/)
  University-level lectures, exam-relevant depth. Use for: theoretical explanations, time complexity proofs.

- [Visualgo](https://visualgo.net/en)
  Interactive animations for DS operations. Use for: understanding traversal, insertions, deletions visually.
